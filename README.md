# ВиК График Генератор

Локално Streamlit приложение за автоматично генериране на строителни линейни графици (Gantt) за ВиК инфраструктурни проекти.

## Стартиране

1. Копирай `.env.example` в `.env` и добави Anthropic API ключа
2. `pip install -r requirements.txt`
3. `streamlit run app.py` (или двоен клик на `start.bat`)

## Работа с приложението

1. В страничната лента посочи път до проектната папка
2. Опиши в чата какъв график ти трябва
3. AI анализира документите и генерира график
4. Прегледай Gantt диаграмата и свали PDF/XML

## Знания

Приложението учи от всеки проект:
- **Уроци** — конкретни случаи и решения
- **Методики** — обобщени правила по тип проект
- **Skills** — техническо ядро (производителности, формули)
