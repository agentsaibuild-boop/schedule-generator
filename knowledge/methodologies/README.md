Те се развиват с времето, базирани на натрупаните уроци.

## Типове:
- engineering_projects.md — ИНЖЕНЕРИНГ (Проектиране + Строителство)
- distribution_network.md — Разпределителна мрежа
- supply_pipeline.md — Довеждащ водопровод
- single_section.md — Единичен участък
