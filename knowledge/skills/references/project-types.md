# Типове проекти

## 1. Разпределителна мрежа (Плевен-type)
- Много еднакви участъци (~199м)
- 10 стандартни дейности на участък
- Pipeline overlap 16 дни
- Fixed days по DN

## 2. Довеждащ водопровод (Харманли-type)
- Променливи участъци (74-354м)
- 7-9 дейности на участък
- Последователно (1 екип)
- Productivities-based

## 3. Единичен участък (Перник-type)
- 1 непрекъснат участък
- FS последователни + FF за design
- Productivities-based

## 4. ИНЖЕНЕРИНГ
- Проектиране + Строителство + Авт. надзор
- 3 фази, 780-1000 дни

## 5. Mega-scale (Враца-type)
- >20км, >1000 задачи
- Само анализ + extract productivities

## 6. Аварийно/HDD (OUT OF SCOPE)
- HDD, microtunneling
- Уникални дейности, <20 дни

(Ще бъде синхронизиран с .claude/skills/construction-schedule/references/)
