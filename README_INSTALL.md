# ВиК График Генератор — Инсталация

## Какво е това?
Приложение за автоматично генериране на строителни графици за ВиК тръжни процедури.

## Инсталация (еднократно, 5 минути)

1. Получете папката с приложението от администратора (USB, споделена папка или ZIP)
2. Разархивирайте (ако е ZIP) на удобно място, напр. `D:\Programs\schedule-generator`
3. Отворете папката и **кликнете двойно** на `install.bat`
4. Изчакайте инсталацията да приключи (3-5 минути)
5. На десктопа ще се появи пряк път **"ВиК Графици"**

## Ежедневна работа

- **Двоен клик** на "ВиК Графици" на десктопа
- Или двоен клик на `start.bat` в папката
- Приложението се отваря в браузъра автоматично
- За спиране: затворете черния прозорец (командна линия)

## Обновяване

- Двоен клик на `update.bat` в папката
- Или получете нова версия от администратора

## Проблеми?

| Проблем | Решение |
|---------|---------|
| "Python не е намерен" | Стартирайте `install.bat` отново |
| Браузърът не се отваря | Отворете ръчно: http://localhost:8501 |
| "Липсва .env" | Стартирайте `install.bat` отново |
| Бавно стартиране | Нормално е при първо пускане (30 сек) |
| Грешка с API | Обадете се на администратора |

## Контакт
При проблеми: обадете се на администратора на системата.
