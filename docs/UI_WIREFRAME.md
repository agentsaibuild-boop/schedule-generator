## 6. Правила за Условна Видимост

| Условие | Елементи | Описание |
|---------|----------|----------|
| `project_loaded` | #2 (Смени проект) | Показва се само когато има зареден проект |
| `!project_loaded` | #3-8 (Load секция) | Скорошни проекти, path input, browse, load, convert |
| `needs_conversion > 0` | #7 (Конвертирай) | Файлове чакат конверсия |
| `converted > 0` | #8 (Преконвертирай) | Вече конвертирани, за повторна конверсия |
| `rollback_dialog` | #11-13 (Admin code) | При натискане на "Върни" |
| `docs_updates_available` | #14 (Обнови) | Когато има нови версии на документацията |
| `restored_history` | #16 (Expander) | Когато има възстановена история от предишна сесия |
| `processing` | #17 (Спри) | Докато AI обработва заявка |
| `schedule_exists` | #31-32 (Gantt, Tabs) | Когато има генериран/демо график |
| `pdf_ready` | #36 (Свали PDF) | След генериране на PDF |
| `xml_ready` | #38 (Свали XML) | След генериране на XML |
| `task_selected` | #40 (Промени в чата) | Когато е избрана задача в Gantt диаграмата |
